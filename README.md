## 📦 Sistema de Controle de Estoque e Financeiro

Desenvolvimento de um sistema para **controle de estoque e organização financeira** de uma microempresa, com foco em praticidade, centralização das informações e apoio à tomada de decisão.

Atuei desde o levantamento das necessidades do negócio até a implementação da solução, incluindo modelagem das entidades, criação do banco de dados com Entity Framework, desenvolvimento de API REST com ASP.NET e interface web utilizando Blazor.

O sistema permite o cadastro e gerenciamento de produtos, controle de entradas e saídas de estoque e acompanhamento financeiro básico.
